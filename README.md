# 수어 인식 시스템
이 프로젝트는 수어 인식 시스템을 위한 데이터 수집, 학습, 평가 및 실시간 제스처 인식을 포함한 완전한 프레임워크를 제공합니다. 각 모듈은 강력한 데이터 수집, 효율적인 모델 학습 및 실시간 웹캠 기반 제스처 인식을 보장하도록 설계되었습니다.

## 주요 기능
### 1. 데이터 수집 모듈 (making_dataset.py)
- MediaPipe를 활용한 손 랜드마크 감지
- 단어 데이터 수집 지원
- 제스처별 폴더에 데이터를 자동으로 정리하고 .npy 파일로 저장
- 특수 제스처 처리 및 동적 사용자 입력 라벨 지정 가능

### 2. 데이터 전처리 모듈 (preprocessing_data.py)
- 모든 제스처 시퀀스를 30 프레임으로 정규화
- 좌우 반전하여 데이터 증강 수행
- 학습 데이터와 검증 데이터를 자동 분할 및 저장

### 3. 모델 학습 모듈 (model_training.py)
- 조기 종료(Early Stopping), 학습률 감소, 모델 체크포인트 저장 기능 포함
- 시퀀스 처리를 위한 슬라이딩 윈도우 기법 적용하여 연산량 감소

### 4. 모델 평가 모듈 (model_test.py & simplemodeltest.py)
- 손실, 정확도, F1-score 등의 학습 진행 과정 시각화
- 저장된 테스트 데이터셋을 이용하여 훈련된 모델 평가 가능

### 5. 실시간 인식 모듈 (model_test_webcam.py)
- 실시간 수어 인식 사용자 인터페이스 제공
- 사용자가 제스처를 연습할 수 있는 퀴즈 모드 포함
- 인식이 잘못되거나 지연될 경우 타이머 기반 피드백 제공
- 인식된 제스처를 확률과 함께 표시하며 사용자의 학습 진행 상황 추적



## 데이터셋 관련 사항
- `dataset/` 폴더에는 원본 데이터셋이 포함되어 있으며, 이미지, JSON 라벨 및 .npy 파일이 있습니다.
